Hướng Dẫn Setup

Kết nối landing page với Google Sheets (lưu data) + EmailJS (gửi email tự động).

Phần 1: Google Sheets — Lưu Data Đăng Ký

  1. Vào Google Sheets → tạo spreadsheet mới tên "Đăng Ký Khoá Học"
  2. Ở dòng 1, ghi header: Tên | SĐT | Email | Khoá học | Giá | Thời gian
  3. Vào menu Extensions → Apps Script
  4. Xoá code mặc định, paste code bên dưới:
function doPost(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var data = JSON.parse(e.postData.contents);

  sheet.appendRow([
    data.name,
    data.phone,
    data.email,
    data.course,
    data.price,
    data.timestamp
  ]);

  return ContentService
    .createTextOutput(JSON.stringify({status: 'ok'}))
    .setMimeType(ContentService.MimeType.JSON);
}
  1. Bấm Deploy → New deployment
  2. Type: chọn "Web app"
  3. Execute as: "Me"
  4. Who has access: "Anyone"
  5. Bấm Deploy → Copy URL (dạng https://script.google.com/macros/s/ABC.../exec)
  6. Paste URL đó vào file index.html, thay YOUR_SCRIPT_ID trong CONFIG
Lưu ý: Lần đầu deploy, Google sẽ hỏi cấp quyền. Bấm "Review Permissions" → chọn tài khoản → "Advanced" → "Go to (unsafe)" → "Allow".

Phần 2: EmailJS — Gửi Email Xác Nhận Tự Động

  1. Vào emailjs.com → Đăng ký free (200 email/tháng)
  2. Email Services → Add New Service → chọn Gmail → kết nối Gmail của anh → Save. Ghi nhớ Service ID
  3. Email Templates → Create New Template
  4. Thiết kế template email:
    • Subject: Xác nhận đăng ký khoá {{course_name}} - AI Academy
    • To Email: {{to_email}}
    • Content (paste vào body):
Xin chào {{to_name}},

Cảm ơn bạn đã đăng ký khoá {{course_name}}!

Thông tin đăng ký:
- Khoá học: {{course_name}}
- Học phí: {{price}}

Bước tiếp theo:
1. Bọn mình sẽ xác nhận thanh toán trong 1-2 giờ
2. Tham gia nhóm Zalo khoá học: {{zalo_link}}
3. Nhận tài liệu pre-course trong nhóm

Nếu có thắc mắc, reply email này hoặc inbox Zalo nhé!

Trân trọng,
AI Academy
  1. Save template → ghi nhớ Template ID
  2. Vào Account → copy Public Key
  3. Mở file index.html, thay 3 giá trị trong CONFIG:
    • EMAILJS_PUBLIC_KEY = Public Key
    • EMAILJS_SERVICE_ID = Service ID
    • EMAILJS_TEMPLATE_ID = Template ID

Phần 3: Link Nhóm Zalo

  1. Mở Zalo → tạo nhóm khoá học (hoặc dùng nhóm có sẵn)
  2. Vào nhóm → Settings → "Link tham gia nhóm" → Copy link
  3. Mở file index.html, thay LINK_NHOM_ZALO trong CONFIG và trong thẻ <a> của trang cảm ơn

Phần 4: Test Thử

  1. Mở index.html trong browser
  2. Điền form → bấm gửi → hiện QR
  3. Bấm "Tôi đã thanh toán"
  4. Kiểm tra:
    • ✅ Google Sheets có dòng mới
    • ✅ Email đến inbox (check spam nếu không thấy)
    • ✅ Trang cảm ơn hiện đúng + link Zalo hoạt động
