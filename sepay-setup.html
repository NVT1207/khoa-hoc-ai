🚀 Setup Auto-Detect Thanh Toán

Khi khách chuyển khoản xong, trang web tự phát hiện và chuyển qua trang cảm ơn. Không cần khách bấm nút.

⚠️ Bảo mật: Không đăng các token/key lên GitHub. Chỉ paste vào Google Apps Script (private).

Bước 1: Deploy Google Apps Script

  1. Vào Google Sheets → Tạo file mới tên "Quang Thuong AI - Data"
  2. Menu Extensions → Apps Script
  3. Xoá code mặc định, paste toàn bộ code từ file google-apps-script.js
  4. Bấm Save (biểu tượng đĩa mềm)
  5. Bấm Deploy → New deployment
    • Type: Web app
    • Execute as: Me
    • Who has access: Anyone
  6. Bấm Deploy → Lần đầu sẽ hỏi cấp quyền → Review Permissions → chọn tài khoản → Advanced → Go to project (unsafe) → Allow
  7. Copy URL Web App (dạng https://script.google.com/macros/s/ABC.../exec)
✅ Ghi nhớ URL này — anh sẽ cần ở Bước 2 và Bước 3.

Bước 2: Gắn URL vào Landing Page

  1. Mở file index.html
  2. Tìm đoạn GOOGLE_SHEET_URL
  3. Thay https://script.google.com/macros/s/YOUR_SCRIPT_ID/exec bằng URL vừa copy
  4. Save → commit → push lên GitHub

Bước 3: Cấu Hình Webhook SePay

  1. Đăng nhập my.sepay.vn
  2. Vào menu Công ty → Cấu hình chung → Webhooks
  3. Bấm Thêm Webhook
  4. Điền:
    • URL: paste URL Apps Script của anh (ở Bước 1)
    • Chỉ cho tài khoản: chọn VietinBank 101872834062
    • Trigger: chọn "Tiền vào tài khoản" (money_in)
    • Method: POST
    • Authorization header: để trống hoặc ghi Apikey ZKB8XAVZ05HNUMO73SXFYWGQ3GILNBVQPCJIKFBUE49H2AADBEVKNAJ1PTSWHWLY
  5. Bấm Lưu

Bước 4: Test

  1. Vào lại my.sepay.vn → Webhooks
  2. Bấm Test webhook → chọn sự kiện mẫu → Send
  3. Mở Google Sheets → kiểm tra sheet Transactions có dòng mới không
  4. Nếu có ✅ → webhook OK
  5. Test thử landing page: đăng ký 1 gói → chuyển khoản 1.000đ (test) → web tự chuyển qua trang cảm ơn

Cách Hoạt Động

Khách điền form → QR với mã đơn hàng duy nhất (QTAI...)
                         ↓
Khách chuyển khoản (nội dung có QTAI...)
                         ↓
VietinBank nhận tiền → SePay phát hiện → gửi webhook
                         ↓
Google Apps Script lưu vào sheet "Transactions"
                         ↓
Landing page đang polling mỗi giây → thấy có giao dịch match orderCode + đủ tiền
                         ↓
Tự động chuyển qua trang cảm ơn + gửi email

Xử Lý Sự Cố

Webhook không chạy?

  • Kiểm tra URL Apps Script: mở trực tiếp trong trình duyệt phải trả về "OK"
  • Kiểm tra quyền deployment: phải là "Anyone"
  • Thử deploy lại: Deploy → Manage deployments → Edit → New version

Web không tự chuyển trang sau khi chuyển khoản?

  • Check Google Sheets tab Transactions xem có giao dịch không
  • Nội dung chuyển khoản phải có đúng mã QTAI... (SePay so khớp theo nội dung)
  • Số tiền chuyển phải ≥ giá gói

Khách chuyển thiếu tiền?

  • Code đang check amount >= expectedAmount → chỉ match khi đủ tiền
  • Nếu khách chuyển thiếu → không tự chuyển trang → anh phải liên hệ xử lý
Chi phí: SePay free tier đủ dùng cho 30+ giao dịch/ngày. Google Apps Script: miễn phí 20.000 request/ngày. Đủ cho giai đoạn đầu.
